Welcome to Finance Tracker!

It looks like this is the first time you are using this tool. Here is a few tips to help you get started:

save You have to press Save manually!
This will likely change in the future.
settings Your data is stored under a data key, displayed on the Settings page. Please write it down somewhere.
If you have already entered data on a different device, use the same data key here.
enhanced_encryption You should set an end-to-end encryption password to make sure only you can access your database.
Do not lose your password! There is no way to access your data if you lose it.
code Finance Tracker is still in early development. If you have questions or suggestions, please contact the developer on GitHub or via email.
